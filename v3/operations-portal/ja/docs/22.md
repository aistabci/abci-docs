# 20. ABCIポイント返却

ホーム画面上部の「ABCIポイント返却」のタブをクリックします。

![[[Screenshot]]](img/22_A.png)

[フォーマットをダウンロード]をクリックし、CSVフォーマットのファイルをダウンロードします。

![[[Screenshot]]](img/22_B.png)

!!! warning
    画面に記載されている「CSVフォーマットの注意点」を確認してください。

CSVフォーマットファイルの入力が完了したら、[ファイルをアップロード]ボタンをクリックします。  
ボタンをクリックすると、ファイル選択のダイアログが表示されるので、アップロードするCSVフォーマットファイルを選択してください。  
また、CSVフォーマットファイルをドラッグ＆ドロップしてアップロードすることもできます。

![[[Screenshot]]](img/22_C.png)
![[[Screenshot]]](img/22_C_2.png)

アップロード後、CSVフォーマットファイルの内容が表示されますので、内容が正しいか確認してください。  
内容に問題なければ[ポイント返却]をクリックします。

![[[Screenshot]]](img/22_D.png)

!!! note
    データが1,000行を超える場合は、1ページにつき最大1,000行まで表示され、ページ分割されます。
    
確認画面の[OK]をクリックします。

![[[Screenshot]]](img/22_E.png)

返却完了画面が表示され、画面左上に処理結果が表示されます。

![[[Screenshot]]](img/22_F.png)

!!! note
    [成功データをコピー(CSV形式)]をクリックすると、返却処理に成功したデータをCSV形式でコピーできます。

!!! note
    画面を初期化したい場合は、[全てクリア]をクリックすることで、初期化できます。

処理結果に「失敗」が含まれる場合は、エラー内容を記載した失敗データが表示されますので、内容を確認してください。

![[[Screenshot]]](img/22_G.png)

!!! note
    [失敗データをコピー(CSV形式)]をクリックすると、返却処理に失敗したデータをCSV形式でコピーできます。
